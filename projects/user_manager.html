User Manager

API python desenvolvida com o objetivo de aprender mais sobre o uso de JWT tokens.

Ver no GitHub

A stack utilizada foi: FastAPI, Sqlachemy, Sqlite, JWT e Pydantic. Aprendi os principais conceitos sobre JWT (Json Web Token), onde um access token é gerado quando um usuário faz login no sistema, e deve ser armazenado no frontend. A cada requisição que exige autenticação, o client deve mandar o JWT token no header. Além disso, esse token possui um tempo de expiração, que é definido pelo desenvolvedor.
Todas as rotas de manipulação dos usuário (Create, Read, Update, Delete), exigem autenticação prévia.

Layered Architecture


User:

routers/ → HTTP layer
services/ → Business logic
repositories/ → Data access layer
schemas/ → Pydantic models
core/ → Database configuration
models/ → SQLAlchemy models

Auth Module

routes/ → Auth HTTP endpoints
services/ → Auth token generation logic
models/ → Token data structures
utils/ → Cryptographic helpers (password hashing and verification)

Endpoints

  • POST /api/auth/token – Realiza autenticação e retorna JWT
  • POST /users – Cria um usuário
  • GET /users – Lista usuários
  • GET /users/{id} – Busca usuário por ID
  • PUT /users/{id} – Atualiza usuário
  • DELETE /users/{id} – Remove usuário
