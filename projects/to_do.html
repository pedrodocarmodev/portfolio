To Do Django

Aplicação CRUD Django criada com o objetivo de criar e gerenciar tarefas diárias.

Ver no GitHub

A stack utilizada foi: Django, SQLite e Bootstrap. Com esse projeto consegui me familiarizar ainda mais com a organização de projetos Django, que seguem a arquitetura MTV. Onde temos M para models, onde definimos as estruturas de dados que serão salvas no banco de dados, T de templates, onde fica a UI do nosso app, com os arquivos .html e o V de Views, onde criamos a conexão entre templates e o banco de dados e realizamos a lógica do negócio. Além disso, aprendi mais sobre os métodos HTTP como GET, POST, PUT e DELETE.

MTV Architecture


templates/ → HTML files
views/ → Business logic
models/ → Data structure

Endpoints

  • / - Home page
  • /deletar-tarefa/{id} - Deleta Tarefa
  • /atualizar-tarefa/{id} - Atualiza Tarefa
  • /admin - Painel Admin
