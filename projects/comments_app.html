Comments Django App

Aplicação Django com o objetivo de simular um blog com posts e comentários.

Ver no GitHub

A stack utilizada foi: Django, SQLite e Bootstrap. Com esse projeto consolidei o processo de criação de aplicativos Django. Aprendi também a modelar entidades no Django utilizando ForeignKey para representar relações one-to-many.

MTV Architecture


templates/ → HTML files
views/ → Business logic
models/ → Data structure

Endpoints

  • / - Home page
  • /post/{post_id} - Acessa Post
  • /add-comentario/{post_id} - Adiciona Comentário
  • /deletar-comentario/{post_id}/{id} - Deletar Comentário
  • /admin - Painel Admin
