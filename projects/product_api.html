Go Products API

API REST desenvolvida em Go para cadastro e consulta de produtos, com arquitetura em camadas e ambiente containerizado.

Ver no GitHub

A stack utilizada foi: Go, Gin, PostgreSQL e Docker. A API foi construída utilizando separação por responsabilidades, organizando camadas de controller, use case e repository.
O projeto também foi containerizado com Docker e Docker Compose, permitindo subir a aplicação junto com o banco PostgreSQL de forma simples.

Arquitetura


cmd/ → Ponto de entrada da aplicação
controllers/ → Camada HTTP
usecases/ → Regras de negócio
repositories/ → Acesso ao banco de dados
database/ → Conexão e configuração do PostgreSQL

Endpoints

  • GET /ping – Health check
  • GET /products – Lista produtos
  • POST /product – Cria produto
  • GET /product/:productId – Busca produto por ID
