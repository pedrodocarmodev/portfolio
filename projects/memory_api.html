Memory API

API python criada para treinos de memorização de números decimais e binários.

Ver no GitHub

A stack utilizada foi: FastAPI, SQLalchemy, SQLite e Pydantic. Para este projeto eu trouxe um dos meus grandes hobbies, a memorização como um esporte. O aplicativo é responsável por criar sequências aleatórias de números decimais ou binários para que o usuário memorize. Após a memorização o usuário envia a resposta, que será corrigida na API, e retornará ao usuário seu resultado e a sequência esperada. Pretendo evoluir ainda mais esta API no futuro, adicionando usuários com autenticação JWT, e estatísticas.
Além disso, criei um template html simples, para possuir uma interface onde o usuário possa realizar o treinamento. Para rodar a aplicação, deve-se ter dois terminais ativos, um rodando a API e o outro o servidor estático para o template HTML.


Layered Architecture


routers/ → HTTP layer
services/ → Business logic
core/ → Database configuration
models/ → SQLAlchemy models e Pydantic Schemas

Endpoints

  • POST /training – Cria um treino
  • GET /training/{id} – Busca por um treino
  • POST /training/{id}/answer – Envia uma resposta
