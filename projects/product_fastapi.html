Product Manager API

API REST para gerenciamento de produtos, criada para praticar desenvolvimento backend com FastAPI e boas práticas básicas.

Ver no GitHub

A stack utilizada foi: FastAPI, SQLAlchemy, SQLite e Pydantic. A API permite criar, listar, atualizar e remover produtos, além de filtrar por categoria.
O banco é criado automaticamente com SQLite e a documentação interativa é gerada pelo próprio FastAPI (Swagger e Redoc). Com esse projeto consolidei ainda mais meus conhecimentos sobre RESTful APIs.


Estrutura do Produto


id → Identificador único
name → Nome do produto (único)
desc → Descrição
stock → Quantidade em estoque
price → Preço
category → Categoria do produto
created_at → Data de criação
updated_at → Data de atualização

Endpoints

  • POST /products – Criar produto
  • GET /products – Listar produtos
  • GET /products/{category} – Filtrar por categoria
  • GET /products/{id} – Buscar por ID
  • PUT /products/{id} – Atualizar produto
  • DELETE /products/{id} – Remover produto
