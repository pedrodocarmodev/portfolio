Password Validation API

API simples para validação de senhas, criada para praticar regras básicas de verificação e uso do FastAPI.

Ver no GitHub

Desenvolvida com FastAPI, a aplicação recebe uma senha via rota e retorna a validação com base em critérios definidos no backend.
